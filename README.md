## 🧰 开源工具

### [skill-optimizer](my-skills/skill-optimizer/) — AI 技能体检工具

给 Claude Code 的 Skill 做诊断。四阶段流水线（形态识别 → 问题分析 → 优化建议 → 执行计划），内置机械审计脚本 + A–K 维度诊断清单。

```bash
# 安装
cp -r my-skills/skill-optimizer ~/.claude/skills/

# 使用（在 Claude Code 对话中）
# "帮我用 skill-optimizer 审查 ~/.claude/skills/my-skill/"
```
